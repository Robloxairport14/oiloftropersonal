Home page

[What is the difference between this website and the bigger competitor versions like Zoom, Skype, Microsoft Teams, Google Voice, Discord, etcetera? Let us just say that metaphorically, this website is like a toy hammer that is safe and helps with “Wack A Moll” or toy nails, while there are adult hammers that are more dangerous to kids without them knowing how to properly use it, the adults are aware with its purpose. Such as in reality, the adults know how to use Zoom, Skype, Microsoft Teams, Google Voice, Discord, etcetera. The adults are more fully aware of using those platforms, but the children might not, just like they did not about the use of cell phones, risk calling 9-1-1 just to see what happens. This website will help keep one specific form of curiosity in a safe way to know how it works so the kids can adapt into adults for Zoom, Skype, Microsoft Teams, Google Voice, Discord, etcetera. Another reason this website is safe like a toy hammer is because anyone who is new to online communication can practice on themselves without the need to talk to strangers. The only way to talk to someone else is if there are two or more number of input devices such as cameras, microphones, or both connected to one computer or desktop.]
